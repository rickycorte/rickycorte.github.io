Getting Started

In this guide you will learn how to set up Build System in your game.

Requirements:

  • Know what components are
  • Create prefabs and add components to them
  • Add and set physics layer

Step # 1

The first thing you have to do is to create a new scene or use an existing one. Create/Prepare also some prefabs that we will use later in Build Items. I'll not show how create them because that's beyond the scope of this guide.
Before we start, note that lines beginning with [S] can be skipped if you already have the base asset and you don't want to create a new one.

Step # 2

Now that you have created your scene and prefabs we can create our player or apply Build System components to it.
Let's start:

  • [S] Create a capsule
  • [S] Set the Main Camera as a child of the object
  • [S] Add Player Controller script to it
  • Add Object Placer component

Do you want to know a cool thing? In like 2 seconds we almost created a player controller that can move around the map and can place objects in the scene!

Step # 3

We have created the player controller but it still not work as expected because it can not place things. To do that we have to do some simple things:

  • [S] Set a prefab in Object To Place field if you don't need the Selection UI or you want to test things
  • Set Ghost material to one of your materials or use the one that comes with this Asset Pack
  • [S] Create a new physics layer
  • [S] Apply it to all the surfaces where you want to place objects
  • Set the value of Ground Layer field with physics layers where you want to place objects. Note: the placement is done on every surface that is in one of the specified layers.

Three more seconds are gone and now you are ready to test it! Hit play and have fun.

Step # 4

Take a break, I know we did a huge work before, come back when you are ready to create object selection UI. Ready? Ok let's go on:

  • Create some Build Items : Right Click in Asset Explorer > Building > Item
  • Create a Build Item Container : Right Click in Asset Explorer > Building > Container

Step # 5

When you you have created enouth Build Items you can start to fill their fields.
Set all the fields of the Items with your prefabs and other data, if your object has multiple mesh renderes check isComplexMesh.

At this point I know you want to ask: "How do I create a object preview?" That's easy, in the top bar, under Build System, click Create Object Preview and follow the instruction of the window that will open.

Step # 6

Now that you have finished filling all the items fields, it's time to add them in the Container. Create new entries with the '+' sign and drag your Build Items. You should end up with something like this (obviusly the names will be different):

Note: to remove and item you can select it and the press '-' of you can just press the 'x' button.

Step # 7

I know you are really tired, but I swear the end of this guide is near.
Let's get back to our player controller:

  • Add Object Selector component
  • Drag and drop the Container that we created before in Build Object List
  • [S] Create your own selector UI prefab by implementing IItemSelectionUI
  • Drag and drop your UI prefab in Builder Menu Prafab field
  • [S] Set a key for menu collapse in the last field
  • Create an Event System in the scene. Right Click in Hierarchy View > UI > Event System

Step # 8

Enjoy! You have added Build System to your game!
