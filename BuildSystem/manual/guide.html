Getting Started

In this guide you will learn how to set up Build System in your scene.

Requirements:

  • Leatest Unity 5 version installed
  • Know what components are
  • Create prefabs and add components to them
  • Add and set physic layer

Step # 1

The first thing you have to do is to create a new scene or use an existing one. Create/Prepare also some prefabs that we will use later in Build Items. I'll not show how create them because that's beyond the scope of this guide.
Before we start note that lines that begins with [S] can be skipped if you already have the base asset and you don't want to create a new one.

Step # 2

Now that you have created your scene and prefabs we can create our player or apply Build System components to it.
Ok lets start:

  • [S] Create a capsule
  • [S] Set the Main Camera as a child of the object
  • [S] Add Player Controller script to it. Wow, in like 2 seconds you have just created a player controller that can move around the map
  • Add Object Placer component

Do you want to know a cool thing? We almost created a player controller that can place objects in the scene!

Step # 3

Ok, we created the player controller but it still not work as expected. We have to do some simple things.

  • [S] Set a prefab in Object To Place field if you don't need the Selection UI
  • Set Ghost material to one of your materials or use the one that comes with this Asset Pack
  • [S] Create a new physic layer
  • [S] Apply it to all the surfaces where you want to place objects
  • Set the value of Ground Layer field with physic layers where you want to place objects. Note: the placement is done on every surface that is in one of the specified layers.

Now you are ready to test it! Hit play and have fun.

Step # 4

Are you ready to continue and create object selection UI? Ok let's go on:

  • Create some Build Items : Right Click in Asset Explorer > Building > Item
  • Create a Build Item Container : Right Click in Asset Explorer > Building > Container

Step # 5

When you are ready we can start to fill Build Items:
Set all the fields in the Item and if your object has multiple mesh renderes check isComplexMesh.

At this point you will want to ask me: "How can I create a object preview? "That's easy, in the top bar, under Build System, click Create Object Preview and follow the instruction of the window that will open.

Step # 6

Now that you have finished filling items fields, it's time to add them in the Container. Create new entries with the '+' sign and drag your Build Items. When you finish you will have something like this:

Note: to remove and item you can select it and the press '-' of you can just press the 'x' button.

Step # 7

I swear the end of this guide is near. Let's get back to our player controller:

  • Add Object Selector componet
  • Drag and drop the Container that we created before in Build Object List
  • [S] Create your own selector UI prefab by implementing IItemSelectionUI
  • Drag and drop your UI prefab in Builder Menu Prafab field
  • [S] Set a key for menu collapse in the last field
  • Create an Event System in the scene. Right Click in Hierarchy view > UI > Event System

Step # 8

Enjoy! You have added Build System to your game!
