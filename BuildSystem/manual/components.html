Component Reference

Object Placer

This component allows you to place one item on the ground.
Object Placer script is the core of the whole Asset Pack, it handles: player input, object placement, rotation, surface alignment, ghost creation and destruction.
This script can run alone if you set all the parameters in the Editor.
Skinned meshes are not supported!

Properties
Object To PlacePrefab to place on the ground. (Leave empty is you are using Object Selector)
Ghost MaterialThe material that will be applied to object ghost.
CamThe camera that will be used to perform the raycast. If empty Camera.main will be used.
Ground LayerLayer where this script can place object.
Max Place DistanceMax distance from Cam where this script can place objects.
Place In Screen CenterPlace object on center of the screen, if false the cursor position is used.
Face MeObject will always face the camera. If false snap rotation will be used.
Snap Rotation AngleSnap angle to add/subtract when a input key is pressed.
Toggle KeyKey used to toggle this script.
Place KeyKey used to place an object.
Positive Rotate KeyKey used to rotate forward the object by Snap Rotation Angle.
Negative Rotate KeyKey used to rotate backward the object by Snap Rotation Angle.

Object Selector

This component handles the object selection UI generation, input and toggles. This script is important because it is the intermediary between UI and Object Placer. I made this choise to grant you an easier UI custumization because you don't have to care about how items are handled by my scripts. By default this script will set Object To Place (Object Placer) to the first element in Build Object List.
NOTE:The toggle key is the same as Object Placer.

Properties
Build Object ListBuild Item Container object that contains the list of spawnable items.
Builder Menu PrefabSelection UI prefab that will be spawned and used to display Build Object List items.
Collapse Menu KeyKey used to collapse Builder Menu Prefab.

Builder UI

This component defines the object selection UI behaviour and then tells Object Selector which object has beed selected.
Note: You can easly create your own selection UI. To create it just implement IItemSelectionUI interface in your script and replace the prefab in Object Selector.

Properties
Button ParentThe transform that will be used as parent for menu entries.
Obj PrefabEntry prefab of type Builder Object UI.
Collapse Menu Button TextReference to the Collapse text in the corresponding UI Button.
Collapse After SelectIf true the menu will be collapsed after an entry is selected.

Builder Object UI

A Builder UI Menu entry spawned at runtime. This component sets up the entry with the item provided at spawn.

Properties
Select ColorBorder color of the entry when is selected.
Not Select ColorBorder color of the entry when is not selected.
Item NameReference to the UI text that displays the item name.
Item ImageReference to the UI image that displays the item preview.
Border ImgReference to the UI image used as border.

Build Item

This script represents an item that can be placed in the scene at runtime. An object of this type can only be created in the editor as a stand alone object and can not be changed at runtime!

Properties
Display NameName that will be showed in the object selection UI.
Prefab To SpawnPrefab that will be spawend in the scene.
Sprite To Show In UIItem preview that will be shown in object selection UI.
isComplexMeshEnable complex mesh ghost creation. Check this only if your mesh has multiple mesh renderes.

Build Item Container

List of Build Items that will be used to generate object selection UI enitres. This items can only be created in the editor as stand alone objects and can not be changed at runtime!

Properties
Build Item ListList of Build Items to use.

Object Preview Generator

Editor tool that helps you to create prefab previews. I reccomend you to create your own previews because this tool generates only low resolution images based on Unity asset previews.

Properties
ObjectPrefab used to create the preview.
